🌟 Personal Portfolio Website

A responsive, modern portfolio built with TypeScript, React, and TailwindCSS. Showcases my projects, skills, experience, and allows visitors to get in touch. Features smooth animations, interactive elements, and a sleek glassmorphic design.

✨ Features

Animated Navigation Bar – Glassmorphic with backdrop blur on scroll
Hero Section – Eye-catching introduction with animated gradient border and bouncing scroll indicator
About Section – Professional bio with stats, info cards, and tech stack badges
Skills Section – Organized tech stack with proficiency bars and color-coded badges
Projects Showcase – Interactive carousel with category filters and project metrics
Services Section – Bento-style feature cards with hover glow effects
Contact Form – Validated form with animated submit button and alerts
Social Media Links – GitHub, LinkedIn, Twitter icons with scale and color animations
Footer – Multi-column layout with quick links and animated heart icon
Scroll Reveal Animations – Smooth fade-in effects as elements enter the viewport

💻 Tech Stack

Frontend: React, TypeScript, TailwindCSS

Animations & Effects: CSS animations, scroll reveal
